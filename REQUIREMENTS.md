# 小朋友兴趣班课时统计 - 手机网页版 Demo

## 产品定位

家庭管理孩子兴趣班课程的工具：管课程、记上课、看日历、算花费。
手机网页版 Demo，数据存 localStorage，不需要后端。

## 技术栈

- React 18 + Vite
- Tailwind CSS（移动端优先）
- Chart.js（饼图、柱状图）
- React Router（底部 Tab 导航）
- localStorage 持久化
- 中文界面

## 视觉风格

- 简洁、儿童友好、马卡龙配色（柔和圆角卡片）
- 每门课程有固定颜色标签（预设 8-10 种颜色循环分配）
- 大字体、大按钮，操作简单，长辈也能用
- 移动端优先，宽度按 375px 设计，最大 480px 居中

## 整体架构

4 个底部 Tab 导航：
1. 首页（课程总览）
2. 课程日历
3. 消费统计
4. 家庭管理

## 数据结构

### Child（孩子）
```
{
  id: string,
  name: string,
  avatar: string (emoji),
  createdAt: timestamp
}
```

### Course（课程）
```
{
  id: string,
  childId: string,
  name: string,           // 课程名称（美术、围棋、游泳等）
  institution: string,    // 机构名称（可选）
  teacher: string,        // 授课老师（可选）
  totalSessions: number,  // 总购买课时
  usedSessions: number,   // 已消耗课时
  sessionsPerClass: number, // 单次上课消耗课时数（默认1）
  totalFee: number,       // 课程总费用
  feePerSession: number,  // 单课时金额（自动计算：totalFee / totalSessions）
  weekdays: number[],     // 每周上课日（0=周日, 1=周一...6=周六，可多选）
  startTime: string,      // 上课开始时间 "HH:mm"
  endTime: string,        // 上课结束时间 "HH:mm"
  color: string,          // 课程颜色标签
  createdAt: timestamp
}
```

### ClassRecord（上课记录）
```
{
  id: string,
  courseId: string,
  childId: string,
  date: string,           // 上课日期 "YYYY-MM-DD"
  sessions: number,       // 本次消耗课时
  fee: number,            // 本次消耗金额（自动计算）
  createdAt: timestamp
}
```

### Family（家庭 - 纯 UI 展示）
```
{
  name: string,
  members: [{ name: string, avatar: string, role: 'admin' | 'member' }]
}
```

## 页面详细需求

### 1. 首页 — 课程总览

**顶部：**
- 孩子头像 + 姓名，横向滑动切换多个孩子
- 右上角 "+" 按钮添加孩子

**课程卡片列表（当前选中孩子的课程）：**
每张卡片显示：
- 左侧颜色条标识
- 课程名称（大字）
- 机构名称 / 老师（小字，可选）
- 课时进度条：已消耗 / 总课时，带百分比
- 剩余课时数字（醒目）
- 每周上课时间：周X HH:mm-HH:mm
- 单课时金额、已消费金额
- 预计课时用完日期（根据每周频率自动推算）
- 操作按钮：【标记上课】【编辑】【删除】

**底部：** 浮动 "添加新课程" 按钮

**标记上课逻辑：**
- 点击 "标记上课" → 弹窗确认（显示将扣减的课时数和金额）
- 确认后：usedSessions += sessionsPerClass，生成一条 ClassRecord
- 支持撤销：最近一条记录可撤销

### 2. 课程日历页

**顶部：** 周视图 / 日视图 切换按钮 + 日期导航（上一周/下一周）

**周视图（默认）：**
- 横向：周一到周日
- 纵向：时间段（8:00-21:00，每小时一行）
- 课程用色块显示：课程名称 + 时间段
- 只显示当前选中孩子的课程

**日视图：**
- 单日时间轴，显示当天所有课程

**点击课程色块弹出详情：**
- 课程名称、老师、机构
- 剩余课时
- 按钮：标记上课 / 查看上课记录

### 3. 消费统计页

**顶部切换：** 今日 / 本月 / 本年 / 全部

**数据卡片：**
- 总消费金额（大数字）
- 总消耗课时
- 课程数量

**图表区：**
- 饼图：各课程消费占比
- 柱状图：月度消费趋势（本年视图）或日消费趋势（本月视图）

**明细列表：**
- 每条记录：日期 + 课程名 + 消耗课时 + 对应金额
- 按时间倒序

### 4. 家庭管理页（纯 UI）

- 家庭名称（可编辑）
- 家庭成员列表：头像 + 昵称 + 角色（管理员/成员）
- "邀请家人" 按钮 → 展示一个模拟邀请码
- "移除成员" 按钮（管理员才有）
- 权限说明文字：管理员可增删改课程，成员可查看和标记上课
- 底部：个人信息 / 退出登录（纯 UI）

### 5. 二级页面

**添加/编辑课程页面：**
- 全字段表单（课程名、机构、老师、总课时、总费用、每次消耗课时、上课星期、开始结束时间）
- 表单验证：课程名和总课时必填
- 费用可不填（默认0）

**上课历史记录页：**
- 某门课程的所有上课记录列表
- 每条：日期 + 消耗课时 + 金额
- 最近一条可撤销

## 自动计算逻辑

1. **剩余课时** = totalSessions - usedSessions
2. **单课时金额** = totalFee / totalSessions（totalFee > 0 时）
3. **已消费金额** = usedSessions × feePerSession
4. **预计用完日期** = 剩余课时 ÷ (每周上课次数 × sessionsPerClass) × 7天，从今天算起
